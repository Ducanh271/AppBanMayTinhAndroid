📱 AppBanMayTinhAndroid
🚀 Giới thiệu

AppBanMayTinhAndroid là ứng dụng mobile mô phỏng hệ thống bán máy tính, cho phép người dùng duyệt sản phẩm và xem chi tiết.

Dự án nhằm демонстра kỹ năng phát triển Android, bao gồm thiết kế giao diện, xử lý dữ liệu và xây dựng luồng ứng dụng thực tế.

✨ Tính năng chính

🛒 Hiển thị danh sách sản phẩm máy tính

🔍 Xem chi tiết sản phẩm (tên, giá, mô tả, hình ảnh)

📂 Giao diện danh sách rõ ràng, dễ sử dụng

🌐 Lấy dữ liệu từ API hoặc nguồn local

📱 UI thân thiện, tối ưu cho mobile

🧠 Điểm nổi bật

Xây dựng luồng e-commerce cơ bản trên mobile

Tổ chức code theo hướng dễ mở rộng và bảo trì

Áp dụng các thành phần phổ biến trong Android (Adapter, Model, UI)

Mô phỏng một ứng dụng thực tế thay vì demo đơn giản

🛠️ Công nghệ sử dụng

Kotlin / Java (Android)

Android Studio

XML / Jetpack Compose

RecyclerView / LazyColumn

REST API / Local Data
